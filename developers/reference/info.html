Info¶

This module provides simple functions for users to retrieve system information.

info.distribution()¶

Returns the current running distribution.

Returns:str -- Returns the first line of /etc/*-release
info.hostname()¶

Returns the host name of the device.

Returns:str -- The devices host name
info.is_64bit()¶

Returns if the device is running a 64bit os.

Returns:bool -- True, if platform is 64 bit
info.kernel()¶

Returns the current kernel release of the device.

The kernel release tells you what version of the kernel is currently used.

Returns:str -- The kernel release of the device
info.pkg_installed(pkg_name)¶

Returns if the given package is installed on the device.

Parameters:pkg_name (str) -- The name of a package
Returns:bool -- True, if installed
info.username()¶

Returns the username of the user that executed uberdot.

Returns:str -- Username of current user
