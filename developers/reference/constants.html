constants.VERSION = '1.10.2_3'¶

Version numbers, seperated by underscore.

First part is the version of Dotmanager. The second part (after the underscore) is the version of the installed-file schema. The latter will be used to determine compability of the Dotmanager with the installed-file.

constants.loadconfig(config_file, installed_filename='default')¶

Loads constants from the config files.

This will load all configs from CFG_FILES plus config_file if provided. The name of the installed-file will be used to load installed-file specific values.

Parameters

  • config_file (str) – Path to the additional config file. If None, only the config from CFG_FILES will be loaded.

  • installed_filename (str) – Name of the installed-file for that values will be loaded
