Differencelog¶

This module implements the Difference-Log

class differencelog.DiffLog(data=None)¶

This class holds the DiffLogData and provides helpers to create and insert DiffLogOperations. Furthermore it provides run_interpreter() that allows to run several interpreter at the same time
