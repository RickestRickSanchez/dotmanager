Profile¶

This module implements the superclass for all profiles

class profile.Profile(options=None, directory=None, parent=None)¶

This class provides the “API” for creating links. It is also responsible for running a profile

Creates a link entry for current options and a given target. Also lets you set the dir like cd or options temporarily only for a link

Create function that can lookup variables set in the profiles

Raise a GenerationError with the profilename

Restores old Builtin mappings

_Profile__set_builtins()¶

Maps functions from self to builtins, so profiles don’t need to use “self” everytime
