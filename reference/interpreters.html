Interpreters¶

This module contains all the different Interpreters. Interpreters implement the behavior for all or a subset of DiffOperations. This could be: - Checking if DiffOperations contradict each other - Printing a DiffOperation - Gaining root access if a DiffOperation needs them - Actually create links according to the DiffOperation
